⚡ FIREBASE SETUP
Connect a free Firebase project to enable cross-device accounts, ratings, reviews, and threaded replies.
1. Go to console.firebase.google.com and create a project
2. Click Add app → Web, register it, copy the firebaseConfig object
3. In the console, enable Authentication → Email/Password
4. Enable Firestore Database → Start in test mode (30-day open access)
5. Paste your config JSON below and click Connect:
Config is stored in this browser's localStorage. For production, update your Firestore Security Rules before going live.
